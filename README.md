# Твой ритм

Твой ритм — сервис-помощник в правильном распределении вашего времени и управлении задач.

## Требования
Для развертки сервиса требуется готовая модель ИИ в формате .gguf (Мы использовали Meta-Llama-3.1-8B-Instruct-Q4_K_M.gguf из 
[bartowski/Meta-Llama-3.1-8B-Instruct-GGUF](https://huggingface.co/bartowski/Meta-Llama-3.1-8B-Instruct-GGUF). Модель нужно поместить в models/ (относительно корня проекта) и назвать **kokos.gguf**
Кроме того, требуется интерпретатор Python (версии не менее 3.14), git и docker

## Запуск
Для начала заполните настройки серверов в **.env** файле в корне проекта:
- WEB_SECRET_KEY - ключ секретного шифрования для веб-сервера
- AI_HOST - адрес сервера для работы с ИИ (по умолчанию localhost)
- DISABLE_AI - отключает все ИИ сервисы, если значение > 0 (ИИ-запросы все еще можно будет сделать, но никакого результата не будет)

Соберите изображение docker и создайте контейнер из него
```bash
git clone https://github.com/rol506/yourRhythm
cd yourRhythm
docker build -t your-rhythm .
docker run -p 4221:4221 your-rhythm
```

После этого веб-интерфейс будет доступен по **http://127.0.0.1:4221>/**
