# Твой ритм

Твой ритм — сервис-помощник в правильном распределении вашего времени и управлении задач.

## Архитектура
Сервис состоит из 2-х серверов:
- Основной веб-сервер
- Сервер для работы с ИИ

## Требования
Для развертки сервиса требуется готовая модель ИИ в формате .gguf (Мы использовали Meta-Llama-3.1-8B-Instruct-Q4_K_M.gguf из 
[bartowski/Meta-Llama-3.1-8B-Instruct-GGUF](https://huggingface.co/bartowski/Meta-Llama-3.1-8B-Instruct-GGUF)). Модель нужно поместить в models/ (относительно корня проекта) и назвать **kokos.gguf**
Кроме того, требуется интерпретатор Python (версии не менее 3.14) и git.

## Запуск
Для начала заполните настройки серверов в **.env** файле в корне проекта:
- WEB_SECRET_KEY - ключ секретного шифрования для веб-сервера
- WEB_PORT - порт, на котором будет открыт веб-интерфейс (по умолчанию 4221. Для prod-развертывания рекомендуем 80)
- AI_HOST - адрес сервера для работы с ИИ (по умолчанию localhost)
- AI_PORT - порт сервера для работы с ИИ (по умолчанию 4222)

Далее создайте виртуальную среду (venv) и установите все зависимости:
```bash
git clone https://github.com/rol506/yourRhythm
python -m venv yourRhythm
cd yourRhythm
source bin/activate # Scripts/Activate.bat на Windows
pip install -r requirements.txt
```

Теперь запустите 2 веб сервера командами:
- ```python src/flsite.py``` - основной веб-сервер.
- ```python src/AIServer.py``` - сервер для работы с ИИ.

После этого веб-интерфейс будет доступен по **http://127.0.0.1:<указанный в WEB_PORT порт>/**
